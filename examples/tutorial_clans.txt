# Clan file for the Clann tutorial mammal dataset
# (tutorial_single.ph / bsweight_demo.ph)
#
# Format: one clan per line, taxon names space-separated.
# Lines starting with '#' are comments; blank lines are ignored.
#
# A clan is a set of taxa expected to be monophyletic (form a single
# connected subtree) in every correctly inferred gene tree, regardless
# of where the tree is rooted.  Gene trees that violate a clan are
# likely to contain hidden paralogy, mis-assigned orthologs, or
# reconstruction artefacts.
#
# Clans with fewer than 2 members present in a given gene tree are
# skipped as untestable.  Clans where all tree taxa are members are
# skipped as trivially satisfied.
# -----------------------------------------------------------------------
